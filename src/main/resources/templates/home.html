Selamat datang di SI-RUANGAN

SI-RUANGAN adalah aplikasi bagi anggota dari SMA Rajin dan Tekun (SMART) untuk melakukan peminjaman fasilitas

Milestone 1

  • Halaman Utama
  • Menambah Fasilitas
  • Menampilkan Detail Ruangan
  • Mengajukan Peminjaman Ruangan
  • Mengajukan Pengadaan Fasilitas

Milestone 2

  • Log in dan Log out
  • Daftar Fasilitas untuk SI-Koperasi
  • Membuat Pengadaan Buku ke SI-Perpustakaan
  • Menampilkan User Profie
  • Menambah User

Milestone 3

  • Menampilkan Daftar Pengajuan Pengadaan Fasilitas
  • Menghapus Pengajuan Pengadaan Fasilitas
  • Mengubah Jumlah Fasilitas
  • Menghapus Fasilitas
  • Menampilkan Daftar Pengajuan Peminjaman Ruangan
  • Mengubah Persetujuan Peminjaman Ruangan
